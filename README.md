# Diabetic Prediction Using Deep Learning

This project predicts diabetes using a deep learning model (ANN) trained on medical data.

## Technologies
- Python
- TensorFlow / Keras
- Pandas, NumPy
- Scikit-learn

## Dataset
PIMA Indians Diabetes Dataset

## Model
Artificial Neural Network (ANN)

## How to Run
1. Install requirements  
   pip install -r requirements.txt
2. Run the model  
   python diabetic_prediction_ann.py

## Output
Predicts whether a patient is diabetic or non-diabetic.
